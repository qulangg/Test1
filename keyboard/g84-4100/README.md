# G84-4100 Based on LKB-Core

G84-4100/4125 系列改装固件。

连线说明：
- P09对应第一个脚
- 跳过P20和P30对应的两根线，将剩下的向前移动两位
- 最后一根线连接到3V3。若需要的话，可以串联一个灯阻。

```
键盘排线  1   2   3  ... 16  17  18  19  ... 27  28
模块引脚 P09 P08 P07 ... P19 NC  NC  P20 ... P21 3V3
(NC) 表示空置，不连接。
```

LED说明：
- 原有的Numlock变为蓝牙指示灯
- 原有的ScrollLock变为USB指示灯

按键说明：
- Fn+Alt：GUI（win）
- Fn+Tab：切换USB
- Fn+CapsLock：睡眠
- 其他功能同原本按键
- Space+U：开机
- Space+U+E：开机并删除绑定
